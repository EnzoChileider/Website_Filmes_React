Para o Correto Funcionamento do aplicativo favor instalar tais frameworks :
Lottie para animações de ícones :
npm install lottie-web
Typewriter para animação de texto : 
npm i typewriter-effect
npm install -D tailwindcss


Para iniciar deve usar 
npm start 
npx json-server --watch database/db.json --port 8000

Preferencialmente usar a extensão LottieFiles for VS Code, se quiser explorar mais o código e executar alguns
testes.
